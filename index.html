Spread Guides — วิธีการวางไพ่
① Single Card — Message
Card 1พลังงานของช่วงเวลานี้ / สิ่งที่ควรรับรู้ / คำแนะนำสั้น ๆ
สูตร: ความหมาย = เลข + ดอก
② Past · Present · Future
Pastสาเหตุของเรื่อง — root of the situation
Presentสถานการณ์ตอนนี้ — current state
Futureแนวโน้มถ้าอะไรยังดำเนินแบบเดิม
เทคนิค: ดูการเปลี่ยนของตัวเลข เช่น 4→7→9 = จากความมั่นคง → มีการทดสอบ → สู่ผลลัพธ์
③ Situation · Challenge · Outcome
Situationภาพรวมของเรื่อง — the full picture
Challengeสิ่งที่ขวางหรือทำให้ยาก — the obstacle
Outcomeผลถ้ารับมือสถานการณ์นี้ — resolution
เทคนิค: ดูว่า Challenge ขัดกับ Situation ไหม
เช่น 6♥ (สมดุลด้านความรู้สึก) + 5♠ (ปัญหาซ่อนอยู่) = ความสัมพันธ์ที่เหมือนจะดี แต่มีความขัดแย้งซ่อนอยู่
④ Path A · Path B
Path Aพลังงานของเส้นทางแรก
Path Bพลังงานของเส้นทางที่สอง — เปรียบเทียบ ไม่ใช่ดูอนาคต
เทคนิค: ตัวเลขใหญ่กว่า = พลังงานแรงกว่า · ดอกบอกลักษณะของทาง
เช่น 8♦ (ต้องทำงานหนัก แต่มีโอกาส) vs 6♥ (สบายกว่า สมดุลกว่า)
